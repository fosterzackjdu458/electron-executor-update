## Overview

Electron Executor is a Roblox script executor for Windows packaged as a lightweight Electron desktop app. It focuses on Lua/script execution with a straightforward desktop workflow, so scripts can be loaded and run from a local application instead of a noisy browser-based setup.

The project also includes a built-in script hub and auto-update support, helping the app stay in step with newer releases and current Roblox scripting demands. If you want a Windows Roblox Lua executor with batch execution, persistent queue handling, and a multi-language interface, Electron Executor belongs in that category.
